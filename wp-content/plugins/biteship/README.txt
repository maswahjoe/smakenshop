== Description ==

Biteship is a plugin for WooCommerce users to enhance their buying experience by adding <b>multiple shipment solutions</b> to their website.

Biteship has partnered over 20+ third party logistics across Indonesia and we're on our way to add more. 

<h3>Features</h3>

* Check rate from multiple couriers
* Check rate for instant couriers like Gojek, Grab, Deliveree and more
* Add multiple pickup points to customize your needs
* Add default weight to your shipment
* Activate only courier services based on your customer needs
* Integrate Biteship web to WooCommerce

== Installation ==

<h3>before you start</h3>

Here are some things you need to know before using the plugin. 
- This plugin requires you to have the <a href="https://woocommerce.com/" rel="nofollow ugc">WooCommerce plugin</a> already installed and activated in WordPress.
- We recommend you use this plugin in a staging environment before installing it on production servers. To learn more about staging environments, <a href="https://wordpress.org/plugins/search.php?q=staging">check out these related WordPress plugins</a>

<h3>Getting Started</h3>
Please follow instructions bellow to connect your WooCommerce to Biteship.

1. Download the plugin.
2. Install the plugin on your WordPress Admin site. 
3. Generate your Biteship key through <a href="https://biteship.com">Biteship Dashboard</a>
4. This will be your free license and couriers will be limited
5. Add your key to the shipment page on your WooCommerce setting > shipping > Biteship page
6. Set new initial address for your shipment origin and click Save Address
7. Don't forget to save if you make any changes at the bottom of your admin page
8. List of couriers available will show up after you have saved your license key in admin page
9. Contact our admin at info@biteship.com if you want to unlock more couriers with paid Biteship License
10. Off you go!

If you need help, please do not hesitant to contact us! info@biteship.com
